\doxysubsection{Detailed Description}
Static methods for managing the hash values used in the library system. 

The {\ttfamily \doxylink{class_library_hash}{Library\+Hash}} class contains static methods that handle the hashing logic used to efficiently store and retrieve books in the library inventory. It provides methods for hashing a book, converting an ISBN to a hash value, and formatting an ISBN string to a standard numeric format.
