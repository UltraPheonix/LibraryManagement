\doxysubsection{Detailed Description}
Manages the library\textquotesingle{}s book inventory, checkout system, reservations, and overdue books. 

The {\ttfamily \doxylink{class_librarian}{Librarian}} class provides functionality for managing books in a library system. It handles checking out books, returning them, managing reservations, processing overdue books, and calculating fines. It also allows searching, adding, and removing books from the library\textquotesingle{}s inventory. The class tracks the availability of books and ensures that books are properly checked out or reserved.
